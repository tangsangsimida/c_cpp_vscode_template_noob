# 示例项目库 (Examples Repository)

欢迎来到项目示例库。本指南旨在通过实战案例深入剖析 C 和 C++ 的核心机制、常见陷阱及应用场景，所有示例均按语言种类进行分类。

## 目录结构

- **[C 语言示例](./C)**: 涵盖 C 语言核心知识点总结、常见坑演示及应用场景剖析 (基于 C99/C11 标准)。
- **[C++ 示例](./CPP)**: (正在规划中) 将包含现代 C++ 特性、类与对象、模板编程等进阶内容。

## 如何使用

1. **进入分类目录**：根据您的学习需求，进入对应的 `C` 或 `CPP` 文件夹。
2. **独立编译运行**：每个具体的示例（如 `example/C/07_pointers`）都是一个**独立的 VS Code + CMake 工程**。
3. **打开文件夹**：在 VS Code 中点击 `文件 -> 打开文件夹`，直接指向具体的示例目录（而不是这里），即可实现“一键构建与调试”。

---
*本项目示例深度解耦，您可以随时删除 `example` 文件夹而不会影响主模板项目的正常使用。*
